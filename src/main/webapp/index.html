Hibernate CRUD

A Simple CRUD Application developed using Hibernate and PostgreSQL

API Endpoints

You can use the following exposed API endpoints to use and experiment on
CRUD operations

  • [GET] /api/students : to read all student objects
  • [GET] /api/students/{id} : to read a particular student object
  • [POST] /api/students : to save a student object
  • [PUT] /api/students/{id} : to update a student object
  • [DELETE] /api/students/{id} : to delete a student object
